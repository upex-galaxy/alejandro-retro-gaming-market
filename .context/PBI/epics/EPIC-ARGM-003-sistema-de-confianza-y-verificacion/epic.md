# Epic: Sistema de Confianza y Verificación

**ID:** EPIC-ARGM-003
**Título:** Sistema de Confianza y Verificación
**Prioridad:** HIGH
**Descripción:** Incluye la verificación de vendedores, el sistema de valoraciones y reseñas, y la gestión de disputas.

## Objetivos

*   Aumentar la confianza entre compradores y vendedores a través de procesos de verificación.
*   Proporcionar un sistema transparente para que los usuarios valoren y reseñen transacciones.
*   Establecer un mecanismo para resolver disputas de manera justa y eficiente.

*   [STORY-ARGM-009: Proceso de verificación para nuevos vendedores](stories/STORY-ARGM-009-proceso-de-verificacion-para-nuevos-vendedores/story.md)

*   [STORY-ARGM-011: Visualización de valoraciones y reseñas recibidas](stories/STORY-ARGM-011-visualizacion-de-valoraciones-y-resenas-recibidas/story.md)
*   [STORY-ARGM-012: Sistema de disputa de transacciones](stories/STORY-ARGM-012-sistema-de-disputa-de-transacciones/story.md)

## Criterios de Aceptación del Épico

*   Los vendedores verificados son claramente identificables.
*   Los usuarios pueden dejar y visualizar valoraciones y reseñas de forma efectiva.
*   Existe un proceso claro y funcional para la resolución de disputas.

## Dependencias

*   EPIC-ARGM-001: Gestión de Cuentas y Perfiles (para la identificación de usuarios).
*   EPIC-ARGM-002: Marketplace: Compra y Venta (para las transacciones que generan valoraciones y disputas).

## Notas Adicionales

*   Considerar la integración con servicios de verificación de identidad de terceros.
*   Establecer directrices claras para el contenido de las reseñas y un sistema de moderación.

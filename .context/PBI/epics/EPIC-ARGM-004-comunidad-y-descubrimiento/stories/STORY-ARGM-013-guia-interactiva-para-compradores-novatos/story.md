# User Story: Guía interactiva para compradores novatos

**ID:** STORY-ARGM-013
**Título:** Guía interactiva para compradores novatos
**Épico:** EPIC-ARGM-004: Comunidad y Descubrimiento
**Prioridad:** MEDIUM

## Descripción

Como comprador novato (Sofía), quiero acceder a una "Guía para Principiantes" interactiva que me haga preguntas sobre mis gustos en juegos modernos para recomendarme una consola retro ideal para empezar.

## Criterios de Aceptación

*   Debe existir un acceso visible a la "Guía para Principiantes" desde la página principal o una sección de ayuda.
*   La guía debe presentar una serie de preguntas interactivas sobre las preferencias del usuario (ej. tipo de juegos modernos que le gustan, géneros preferidos, nivel de dificultad).
*   Basado en las respuestas, la guía debe generar una o varias recomendaciones de consolas retro que se ajusten a los gustos del usuario.
*   Cada recomendación debe incluir:
    *   Nombre de la consola.
    *   Breve descripción de por qué se recomienda (basado en las respuestas del usuario).
    *   Enlace a productos relacionados en el marketplace.
*   El usuario debe poder reiniciar la guía o ajustar sus respuestas para obtener diferentes recomendaciones.
*   La guía debe ser amigable y fácil de usar para usuarios sin experiencia en juegos retro.

## Dependencias

*   Base de datos de productos y consolas retro con sus características.
*   Algoritmo de recomendación basado en preferencias.

## Notas Adicionales

*   Considerar la posibilidad de guardar las preferencias del usuario para futuras recomendaciones.
*   Diseñar la interfaz de la guía de forma atractiva y gamificada.
